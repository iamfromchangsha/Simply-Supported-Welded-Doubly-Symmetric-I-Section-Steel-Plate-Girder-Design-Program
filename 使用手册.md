# 钢板梁设计计算程序 — 使用手册

## 1 程序概述

本程序用于 **35m 简支焊接双轴对称工字形钢板梁** 的设计计算与验算，依据 JTG D64-2015《公路钢结构桥梁设计规范》和 JTG D60-2015《公路桥涵设计通用规范》编写。提供参数输入、截面设计、计算验算、HTML/PDF 报告导出及 SVG 示意图生成等功能。

### 1.1 主要功能

- 荷载计算（一期恒载、二期恒载、公路‑I 级活载、荷载组合）
- 截面自动估算与手动输入
- 双轴对称工字形截面几何特性计算
- 弯曲正应力、剪应力、折算应力验算
- 整体稳定性验算（成桥工况，混凝土桥面板提供连续侧向约束）
- 翼缘局部稳定性验算（b₁/t_f ≤ 15√(235/f_y)）
- 腹板横向加劲肋设计（间距、刚度、截面尺寸）
- 翼缘与腹板连接角焊缝计算（E50 焊条）
- 疲劳强度验算（疲劳荷载模型Ⅰ + Ⅱ）
- 挠度验算（总挠度 ≤ L/500，活载挠度 ≤ L/600）
- 变截面设计（跨中段与支点段变更翼缘尺寸）
- HTML 报告（含 MathJax 渲染的 LaTeX 公式）与 PDF 导出
- SVG 示意图：横截面图、弯矩图、剪力图、变截面布置图、加劲肋布置图

### 1.2 适用规范

| 规范 | 内容 |
|------|------|
| JTG D64-2015 | 强度、稳定、疲劳、挠度、构造要求 |
| JTG D60-2015 | 荷载取值、荷载组合 |

## 2 安装与运行

### 2.1 运行条件

| 环境 | 要求 |
|------|------|
| 操作系统 | Windows 7 / 10 / 11 |
| Python | 3.8 ~ 3.11 |
| 第三方库 | PySide6 ≥ 6.5.0 |

### 2.2 从源代码运行

```bash
pip install PySide6
python main.py
```

### 2.3 从安装包运行

确保已安装 **钢板梁设计计算_安装程序.exe**：

1. 运行 **钢板梁设计计算_安装程序.exe**
2. 按照安装向导指定安装位置（默认 `C:\Program Files\钢板梁设计计算程序`）
3. 勾选"创建桌面快捷方式"
4. 安装完成后，勾选"运行钢板梁设计计算程序"立即启动，或从桌面快捷方式/开始菜单启动

### 2.4 从单文件 EXE 运行

**dist/钢板梁设计计算.exe** 为单文件便携版，直接双击即可运行，无需安装。

> **注意**：首次运行可能被 Windows Defender 或第三方杀软拦截，请选择"仍要运行"或添加信任。

## 3 界面说明

程序主窗口分为 3 个选项卡，采用深色渐变主题。

### 3.1 选项卡一：设计参数

在此输入桥梁的基本几何参数和荷载系数。

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 计算跨径 L | 桥梁的计算跨度 (m) | 35.0 m |
| 桥面板宽度 B | 混凝土桥面板宽度 (m) | 1.8 m |
| 混凝土桥面板厚度 H₁ | 桥面板混凝土层厚度 (m) | 0.08 m |
| 沥青铺装层厚度 H₂ | 桥面沥青层厚度 (m) | 0.06 m |
| 横向分布系数 η | 活载分配到单根主梁的比例 | 0.43 |
| 恒载分项系数 γ_G | 永久荷载分项系数 | 1.2 |
| 活载分项系数 γ_Q | 可变荷载分项系数 | 1.4 |
| 结构重要性系数 γ₀ | 结构重要性调整系数 | 1.1 |
| 活载冲击系数 1+μ | 活载动力放大系数 | 1.14 |

**使用方法**：双击数值或点击右侧上下箭头调整，完成后进入"截面设计"选项卡。

---

### 3.2 选项卡二：截面设计

提供两种截面设计模式。

**自动估算模式**（推荐）：

1. 确认"自动估算（推荐）"单选按钮已选中
2. 点击"重新估算"按钮，程序根据跨径和荷载自动计算最优截面尺寸
3. 查看弹出的估算结果信息框
4. 如需微调，点击"应用推荐值到输入框"，然后切换到"手动输入"模式修改

**手动输入模式**：

1. 选中"手动输入"
2. 在"跨中段截面尺寸"区域输入腹板高度、腹板厚度、翼缘宽度、翼缘厚度
3. 如有变截面设计，输入"变截面段截面尺寸"的翼缘尺寸
4. 点击"计算截面特性"预览截面参数
5. 点击"应用推荐值到输入框"（需先自动估算）将估算值填入输入框

**截面特性预览区**可查看：

- 跨中段截面参数（总高、面积、惯性矩、截面模量、自重）
- 变截面段截面参数（总高、面积、惯性矩、截面模量、自重）

---

### 3.3 选项卡三：结果与导出

**操作步骤**：

1. 点击 **▶ 开始计算** 执行完整验算
2. 等待计算完成（界面显示"计算中，请稍候..."）
3. 计算完成后弹出结果提示框：
   - "全部验算通过！设计合理"——所有检查项均满足规范
   - "部分验算未通过"——需调整截面后重新计算
4. 结果自动显示在预览区域（含完整计算书及 SVG 图解）
5. 使用下方按钮导出报告：

| 按钮 | 功能 |
|------|------|
| 导出 HTML | 保存为 .html 文件，可用浏览器打开查看（支持 MathJax 渲染公式） |
| 导出 PDF | 保存为 .pdf 文件（需等待 PDF 生成进度条完成） |
| 全部导出 | 将 HTML 保存到指定目录（PDF 需单独导出） |

> **PDF 导出注意事项**：需要网络连接以下载 MathJax CDN 字体，页面布局已预设 25mm 页边距。如导出失败，请检查网络连接后重试。

## 4 计算书内容说明

生成的 HTML/PDF 计算书共 11 章：

| 章节 | 内容 |
|------|------|
| 一、设计基本参数 | 几何参数、荷载参数、材料特性（Q345 钢强度设计值） |
| 二、荷载计算 | 二期恒载、活载标准值（公路‑I 级）、一期恒载 |
| 三、截面尺寸拟定 | 跨中段截面尺寸表、变截面段尺寸表、横截面 SVG 图 |
| 四、内力计算 | 荷载组合计算、弯矩图 + 剪力图（SVG） |
| 五、跨中截面强度及刚度验算 | 弯曲正应力、剪应力、折算应力、整体稳定、翼缘局部稳定、疲劳强度、挠度验算 |
| 六、变截面设计 | 变截面段强度验算、变截面布置示意图（SVG） |
| 七、翼缘焊缝设计 | 角焊缝应力计算 |
| 八、局部稳定设计 | 腹板加劲肋设计（间距、刚度、截面尺寸）、加劲肋布置图（SVG） |
| 九、连接构造说明 | 变截面处连接构造 |
| 十、设计结果汇总 | 全部验算项汇总表（满足/不满足） |
| 十一、结论 | 设计结论 |

每个验算项均标注 **满足** 或 **不满足**，方便判断。

## 6 常见问题

**Q：点击"开始计算"后程序无响应？**

A：计算过程会在后台进行，界面会显示"计算中，请稍候..."，通常 1-2 秒完成。如长时间无响应，请检查输入参数是否合理（如跨径是否在 20-50m 范围内）。

**Q：PDF 导出失败？**

A：PDF 导出依赖 QWebEngineView 的 printToPdf 功能，需要 Internet 连接以下载 MathJax CDN 字体。请检查网络连接，也可以先导出 HTML 后用浏览器"打印→另存为 PDF"。

**Q：自动估算的截面不满意？**

A：切换到"手动输入"模式即可自由调整截面尺寸。建议在自动估算值基础上微调。

**Q：部分验算未通过怎么办？**

A：查看计算书中标注"不满足"的验算项，调整对应的截面尺寸后重新计算。常见调整方向：
- **弯曲应力过大** → 增大 h_w 或 t_f
- **剪应力过大** → 增大 t_w
- **挠度过大** → 增大梁高 h
- **翼缘局部稳定不满足** → 增大 t_f 或减小 b_f
- **疲劳不满足** → 增大翼缘面积

## 7 技术说明

- 程序由 Python 3.11 编写，图形界面使用 PySide6（Qt6 的 Python 绑定）
- HTML 报告使用 MathJax CDN 渲染 LaTeX 公式
- PDF 导出通过 QWebEngineView.printToPdf() 实现
- 所有示意图由 Python 直接生成纯 SVG 代码，无第三方图形库依赖
- 打包为单文件 EXE 使用 PyInstaller（--onefile --noconsole）
- 安装程序使用 Inno Setup 6 制作
